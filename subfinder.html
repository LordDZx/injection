جميع أوامر Subfinder مع الشرح

subfinder -d example.com

يجمع جميع النطاقات الفرعية لموقع معين باستخدام المصادر الافتراضية.

subfinder -d example.com -o subdomains.txt

يحفظ جميع النطاقات الفرعية في ملف نصي.

subfinder -d example.com -sources crtsh,shodan

استخدام مصادر إضافية مثل crt.sh و Shodan للحصول على نطاقات أكثر.

subfinder -dL domains.txt -o results.txt

تشغيل الأداة على قائمة مواقع وحفظ النتائج في ملف نصي.

subfinder -d example.com -o subdomains.txt && cat subdomains.txt | httprobe

تشغيل Subfinder ثم فحص المواقع النشطة باستخدام httprobe.

subfinder -d example.com -o subdomains.txt && cat subdomains.txt | httpx -silent -title -status-code

تشغيل Subfinder ثم استخراج عناوين الصفحات وأكواد الاستجابة.

subfinder -d example.com -silent

إخفاء المخرجات غير المهمة وعرض النطاقات فقط.

subfinder -d example.com -recursive

تفعيل البحث العميق داخل النطاقات الفرعية لاكتشاف نطاقات أكثر.

subfinder -d example.com -nW

تعطيل البحث عن نطاقات Wildcard.

subfinder -d example.com -t 50

تحديد عدد الـ Threads لزيادة سرعة البحث (هنا تم ضبطها على 50).

subfinder -d example.com -v

تفعيل وضع التفاصيل (Verbose Mode) لمعرفة المصادر التي تم استخدامها.

subfinder -d example.com -config /path/to/config.yaml

استخدام ملف إعدادات مخصص لتشغيل الأداة بمصادر وبيانات API مختلفة.
