أوامر Katana مع الشرح

katana -u "http://example.com"

يقوم بجمع جميع الروابط من الموقع المحدد.

katana -u "http://example.com" -o output.txt

حفظ جميع الروابط المستخرجة في ملف نصي.

katana -u "http://example.com" -d 3

تحديد مستوى العمق للبحث (هنا العمق 3 مستويات).

katana -u "http://example.com" --js

استخراج الروابط من ملفات JavaScript المخفية.

katana -u "http://example.com" --proxy "http://127.0.0.1:8080"

تمرير الترافيك عبر BurpSuite أو أي Proxy آخر.

katana -u "http://example.com" -H "User-Agent: Mozilla/5.0"

إضافة Header مخصص عند إرسال الطلبات.

katana -list targets.txt -o results.txt

استخدام قائمة مواقع وتنفيذ البحث عليها جميعًا.

katana -u "http://example.com" -f qurl

استخراج فقط الروابط التي تحتوي على استعلامات (مثل ?id=123).
